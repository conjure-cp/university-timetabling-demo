## Application Preview

### Simple tutorial with Tutorial.json file

- Download Tutorial.json file
- Go to **CREATE** tab
- Click the **IMPORT** button
  - Import the Tuorial.json file
- Go to **SOLUTION** tab
- Click **GET SOLUTION** button
- Done!
